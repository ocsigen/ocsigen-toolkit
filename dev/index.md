
# Ocsigen Toolkit

Ocsigen Toolkit is a set of reusable user-interface widgets for [Eliom](https://ocsigen.org/eliom/) applications. The widgets can be produced on the server or on the client from the same code, which makes them particularly suited to mobile, client-server applications.

Ocsigen Toolkit is part of the [Ocsigen project](https://ocsigen.org).


## Documentation

- [Introduction](./intro.md) — installation, programming style and an overview of the available widgets.
- The server and client API of every widget (use the *Server API* / *Client API* switch to move between the two sides).

## Installation

Ocsigen Toolkit is available via OPAM:

```ocaml
opam install ocsigen-toolkit
```
